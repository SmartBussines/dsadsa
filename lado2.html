Misión: Ofrecer servicios informáticos a personas que necesiten de ya sea una aplicacion, página web u organizar grandes cantidades de datos.

Vision: Ser una empresa que a travez del tiempo va mejorando de acuerdo a sus experiencias para ser reconocidos a nivel estatal dentro el área de la programación.

Servicios
  • Creacion de diagramas de flujo para la resolucion de problemas
  • Creacion de bases de datos para poder manejar informacion de una manera facil e intuitiva
  • Creacion de aplicaciones web que cumplan una o varias tareas
  • Creacion de aplicaciones moviles que cumplan una o varias tareas
  • Creacion de paginas web para publicar lo que el cliente quiera exponer.
Objetivo: Proporcionar servicios informaticos a personas que necesiten ya sea una pagina web, aplicacion o bases de datos con las caracteristicas que defina el cliente sera elaborada mediante la aplicacion que el programador elija y sera ejecutada por el usuario.




Valores
Valor Descripcion
Responsabilidad Cumplir con el trabajo y con todos sus requisitos definidos por el cliente en tiempo y forma
Puntualidad Entregar la pagina web, aplicacion, base de datos o el programa ordenado en tiempo y forma
Comunicacion Algun miembro del equipo tendra comunicacion con el cliente para informes
Respeto Se tendra una sana convivencia con el cliente basandose en el respeto
